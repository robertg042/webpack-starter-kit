Webpack starter kit

This is a starter kit for webpack 3 inspired by wonderful Brad Traversy.

Usage

$ npm install
$ npm start

What's included?

  • babel
  • sass
  • postcss
  • autoprefixer
  • Font Awesome Free, Lato and Open Sans fonts
  • simple web page scaffold
